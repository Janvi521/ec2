# Simple Node Backend (Express + ES Modules)

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

If port `3000` busy ho, to:

```bash
PORT=3001 npm start
```

## Endpoints

- `GET /` → returns `hello`
- `GET /api/janvi` → returns JSON data

## Test (optional)

```bash
curl http://localhost:3000/
curl http://localhost:3000/api/janvi
```
